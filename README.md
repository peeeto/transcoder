# Transcoder

## Transcoder project

```bash
git clone git@github.com:peeeto/transcoder.git # download repository
cd transcoder
mvn install

java -jar target/transcoder-1.2-SNAPSHOT.one-jar.jar

./install.sh # creates link: /bin/transcoder -> transcoder.sh
transcoder & # starts application
```
